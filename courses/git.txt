Name: Git speedup
Description: A collection of usefull git commands to incrase the interact with it locally
Command: git

=

Description: Checkout a remote branch to local editable branch 
Schema: git switch <branch>

=

Description: Stage specific files based on path
Schema: git add (<path>...)

=

Description: Stage all changes for cmmit
Schema: git add (--all | -A)

=

Description: List all the stashed changes
Schema: git stash list

=

Description: Edit previous commit message
Schema: git commit --amend

=

Description: Commit staged changes with a message
Schema: git commit (-m <message>)
